# CLEAR

The `CLEAR` statement empties a LIST, thus deleting all its contents. The LIST itself is not deleted though, and can still be used and filled with new elements after a `CLEAR` statement has been executed.

#### Syntax:

```coffeescript
CLEAR <LIST>
```
